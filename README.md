# Análise Exploratória de Dados de Churn

## Contexto do Projeto

O projeto tem como objetivo analisar a evasão de clientes (churn) em um sistema de ERP (Enterprise Resource Planning), utilizado por pequenas e médias empresas para gerenciar e integrar rotinas administrativas, como emissão de notas fiscais, controle de estoque e gestão financeira.

Para isso, foi realizada uma Análise Exploratória de Dados (EDA), com o intuito de explorar a base de dados, compreender o comportamento das variáveis e identificar padrões associados ao cancelamento do serviço.

Ao longo da análise, são consideradas variáveis categóricas e numéricas, buscando identificar relações e diferenças entre os grupos de clientes que permaneceram e os que cancelaram o serviço.

## Metodologia

A análise foi estruturada nas seguintes etapas:

1. **Exploração Univariada** — análise individual de cada variável, verificando distribuição, frequência e valores ausentes
2. **Exploração Multivariada** — cruzamento das variáveis com a variável alvo (Churn) e análise de correlação entre variáveis numéricas
