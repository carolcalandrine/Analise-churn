# Análise Exploratória de Dados de Churn

## Contexto do Projeto

O produto analisado neste projeto é um sistema de ERP (Enterprise Resource Planning), utilizado por pequenas e médias empresas para gerenciar e integrar rotinas administrativas, como emissão de notas fiscais, controle de estoque e gestão financeira.

Este projeto consiste na realização de uma Análise Exploratória de Dados (EDA) com foco na evasão de clientes (churn), ou seja, na identificação de padrões associados ao cancelamento do serviço. A análise tem como objetivo explorar a base de dados, compreender o comportamento das variáveis e investigar possíveis relações entre características dos clientes e a ocorrência de churn.

Ao longo da EDA, são analisadas variáveis categóricas e numéricas, buscando respostas que ajudem a explicar os fatores que podem ser uma possível influência naa saída de clientes e que possam apoiar decisões estratégicas voltadas à retenção.

### Objetivo
Este projeto tem como objetivo realizar uma análise exploratória de dados (EDA) para investigar a hipótese de aumento na taxa de churn (cancelamento de clientes) em uma plataforma de serviços por assinatura.
